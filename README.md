# IPL Data Analysis üèè

This project performs exploratory data analysis (EDA) on IPL match and
ball-by-ball data to uncover insights about team performance, success
ratios, and batting trends.

## Objectives
- Analyze team-wise match performance
- Calculate success ratios of IPL teams
- Compare batting performances using strike rate and runs
- and many more


## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
IPL Matches and Deliveries dataset sourced from Kaggle.

## Key Insights
- Teams with the highest success ratios across seasons
- Comparison of top batsmen based on strike rate vs innings runs
- Impact of different dismissal types on outcomes
